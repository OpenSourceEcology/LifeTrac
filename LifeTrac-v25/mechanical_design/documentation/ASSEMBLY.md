# LifeTrac v25 - Assembly Instructions

## Safety Information

**⚠️ WARNING:** This machine involves heavy components, high-pressure hydraulics, and electrical systems. Follow all safety procedures:

- Wear appropriate PPE (safety glasses, gloves, steel-toe boots)
- Use proper lifting equipment for heavy components
- Follow hydraulic system safety procedures
- Ensure all welds are performed by qualified personnel
- Test all systems before operation

## Prerequisites

### Tools Required
- MIG or TIG welder
- Angle grinder
- Drill press or hand drill
- Torque wrench
- Socket set (metric and SAE)
- Measuring tape and square
- Level
- Lifting equipment (engine hoist or forklift)
- Hydraulic pressure tester

### Skills Required
- Welding (intermediate level)
- Hydraulic system assembly
- Mechanical assembly
- Basic electrical wiring

## Assembly Overview

**Estimated Time:** 40-60 hours for complete assembly (experienced builder)

**Assembly Sequence:**
1. Base frame construction
2. Wheel mounting and installation
3. Hydraulic system installation
4. Loader arm assembly and mounting
5. Bucket assembly and attachment
6. Electronics installation
7. Optional standing deck installation
8. System testing and calibration

## Phase 1: Base Frame Construction

### Step 1.1: Cut Frame Components

**Parts Needed:** A1 components (square tubing)

1. Cut square tubing to specifications:
   - 2× 1800mm (longitudinal members)
   - 3× 1200mm (cross members)
   - 2× 1000mm (vertical uprights)
   - 1× 1200mm (top cross member)

2. Deburr all cuts and clean surfaces

3. Mark welding locations according to design

**Quality Check:** Verify all cuts are square (90°) and lengths are within ±2mm

### Step 1.2: Weld Base Frame

1. **Lay out bottom frame:**
   - Place two longitudinal members parallel, 1000mm apart
   - Position three cross members: front, center, rear
   - Tack weld all joints
   - Check squareness (measure diagonals - should be equal)

2. **Complete bottom frame welds:**
   - Full penetration welds on all joints
   - Both sides where accessible
   - Grind smooth

3. **Add vertical uprights:**
   - Weld at front corners
   - Ensure vertical (use level)
   - Add temporary bracing if needed

4. **Install top cross member:**
   - Position at top of uprights
   - Tack, check level, then complete welds

**Quality Check:** 
- Frame is square (diagonals equal)
- All joints fully welded
- No warping or twisting

### Step 1.3: Install Wheel Mounting Plates (A2)

**Parts Needed:** 4× wheel mounting plates

1. **Mark mounting positions:**
   - Front left: -500mm X, +700mm Y
   - Front right: +500mm X, +700mm Y
   - Rear left: -500mm X, -700mm Y
   - Rear right: +500mm X, -700mm Y

2. **Attach plates:**
   - Drill pilot holes through frame
   - Install 1/2" bolts (4 per plate)
   - Torque to 80 ft-lbs
   - Alternatively: weld plates directly to frame

**Quality Check:** All plates level and aligned

---

## Phase 2: Wheel Installation

### Step 2.1: Prepare Wheel Assemblies (B1-B4)

**Parts Needed:** 4× complete wheel assemblies

For each wheel:

1. **Install bearings in housings:**
   - Press fit or bolt bearing into housing
   - Apply appropriate grease
   - Install seals

2. **Mount hydraulic motor:**
   - Bolt motor to wheel hub
   - Torque mounting bolts to manufacturer specs
   - Connect motor shaft to wheel hub (key and set screw)

3. **Install wheel on axle:**
   - Slide wheel/motor assembly onto axle
   - Install retaining hardware
   - Check for free rotation

**Quality Check:** Wheels rotate freely, motors securely mounted

### Step 2.2: Install Wheel Assemblies

1. **Position wheel assemblies:**
   - Align with mounting plates
   - Use lifting equipment (wheels are heavy!)
   - Install mounting bolts

2. **Torque mounting bolts:**
   - 1/2" bolts to 80 ft-lbs
   - Star pattern to ensure even pressure

3. **Check alignment:**
   - All wheels should contact flat surface evenly
   - Adjust shims if needed

**Quality Check:** Frame sits level with all wheels on ground

---

## Phase 3: Hydraulic System Installation

### Step 3.1: Install Hydraulic Cylinders

**Lift Cylinders (D1, D2):**

1. **Mount base ends to frame:**
   - Position at marked locations on frame
   - Install 1" pivot pins
   - Add retaining clips

2. **Attach rod ends (temporary):**
   - Will be connected to loader arms later
   - Extend cylinders to mid-stroke for arm installation

**Bucket Cylinders (D3, D4):**
- Set aside for installation after loader arms are mounted

### Step 3.2: Install Hydraulic Hoses

Refer to `../../HYDRAULIC_DIAGRAM.md` for complete hydraulic schematic.

1. **Measure and cut hoses:**
   - Allow extra length for movement
   - Use proper hydraulic hose and fittings

2. **Connect wheel motors:**
   - 2 hoses per motor (supply and return)
   - Route along frame with clamps
   - Keep away from hot or moving parts

3. **Connect cylinders:**
   - 2 hoses per cylinder
   - Allow slack for full range of motion

4. **Install hydraulic valves:**
   - Mount valve bank securely to frame
   - Position for easy access to electronics housing

**Safety Note:** Do not pressurize system until all connections are complete and inspected

---

## Phase 4: Loader Arm Installation

### Step 4.1: Assemble Loader Arms (C1)

For each arm (left and right):

1. **Attach reinforcement plates:**
   - Position 1/4" plate alongside square tubing
   - Weld or bolt in place
   - Ensure pivot holes align

2. **Install pivot bushings:**
   - Press fit bronze bushings at base
   - Ream if necessary for proper fit

3. **Add cylinder mounting lugs:**
   - Weld lugs for cylinder attachment
   - Verify hole alignment

### Step 4.2: Mount Loader Arms

**This is a heavy lift - use proper equipment**

1. **Position arms on uprights:**
   - Align pivot holes
   - Insert 1" pivot pins
   - Add retaining hardware

2. **Connect lift cylinders:**
   - Attach cylinder rod ends to arms
   - Install pivot pins
   - Check for full range of motion

3. **Install cross brace:**
   - Connects left and right arms
   - Provides rigidity
   - Bolt or weld in place

**Quality Check:** Arms should pivot smoothly through full range

### Step 4.3: Install Bucket Attachment (C2)

1. **Mount attachment interface to arm ends:**
   - Position at end of loader arms
   - Bolt securely with 1" bolts

2. **Install bucket cylinders:**
   - Mount cylinder bases to loader arms
   - Rod ends connect to bucket attachment
   - Check range of motion

---

## Phase 5: Bucket Assembly

### Step 5.1: Fabricate Bucket (E1)

1. **Cut plate steel components:**
   - Bottom: 1100mm × 600mm
   - Back: 1100mm × 400mm
   - Sides: 600mm × 400mm (2×)

2. **Round all corners:**
   - Use plasma cutter or grinder
   - 6.35mm radius minimum

3. **Weld bucket:**
   - Tack all pieces in position
   - Check angles and fit
   - Complete all welds
   - Grind smooth

4. **Add reinforcement:**
   - Weld reinforcement ribs if needed
   - Bottom wear plate (optional but recommended)

### Step 5.2: Attach Bucket

1. **Position bucket on attachment interface:**
   - Align quick-attach pins
   - May need assistance or lifting equipment

2. **Install attachment pins:**
   - Insert 1" pins
   - Add retaining hardware

3. **Connect bucket cylinders:**
   - Rod ends attach to bucket
   - Install pivot pins

**Quality Check:** Bucket should tilt smoothly through full range

---

## Phase 6: Electronics Installation

### Step 6.1: Install Control Housing (G1)

1. **Mount housing to frame:**
   - Center position for weight distribution
   - Bolt to frame with 1/2" bolts
   - Ensure weatherproof seal

2. **Install electronics:**
   - Follow wiring diagram in `../../WIRING_DIAGRAM.md`
   - Arduino Opta controller
   - Valve controllers
   - Wire connections

3. **Route wiring:**
   - Use cable management
   - Protect from wear and weather
   - Label all connections

**Refer to:** `../../INSTALLATION_GUIDE.md` for complete electronics setup

---

## Phase 7: Optional Standing Deck (F1)

### Step 7.1: Fabricate Deck

1. **Cut deck plate:**
   - 1000mm × 400mm × 6.35mm plate
   - Round all corners

2. **Create anti-slip pattern:**
   - Drill or plasma cut 20mm holes
   - 100mm spacing in grid pattern

3. **Fabricate support brackets:**
   - 2× from 2"×2" square tubing
   - Weld mounting tabs

### Step 7.2: Install Deck

1. **Position at rear of frame:**
   - Align with rear cross member
   - Height for easy step-up (~200mm)

2. **Attach supports:**
   - Bolt or weld to frame
   - Ensure secure mounting

3. **Bolt deck to supports:**
   - Use 1/2" bolts
   - Check that deck is level and secure

## Phase 8: System Testing

### Step 8.1: Pre-Operation Checks

**Mechanical:**
- [ ] All bolts torqued to specification
- [ ] All welds complete and inspected
- [ ] No sharp edges or protrusions
- [ ] All moving parts operate smoothly
- [ ] Frame is square and level

**Hydraulic:**
- [ ] All hoses properly connected
- [ ] Fittings tight (but not over-tightened)
- [ ] Hydraulic fluid at proper level
- [ ] No visible leaks
- [ ] Relief valves set correctly

**Electrical:**
- [ ] All wiring connections secure
- [ ] Controller powered and responding
- [ ] Emergency stop functional
- [ ] Wireless communication working

### Step 8.2: Hydraulic Pressure Test

**⚠️ DANGER: High Pressure System**

1. **Initial fill:**
   - Fill hydraulic reservoir
   - Run pump to circulate fluid
   - Purge air from system

2. **Pressure test:**
   - Gradually increase pressure
   - Inspect all connections for leaks
   - Check pressure gauges
   - Verify relief valve operation

3. **Function test (no load):**
   - Test each cylinder independently
   - Test wheel motors
   - Check for unusual noises or vibrations

### Step 8.3: Functional Testing

1. **Loader arm test:**
   - Raise and lower arms slowly
   - Check for smooth operation
   - Verify full range of motion
   - No binding or unusual noises

2. **Bucket test:**
   - Tilt bucket through full range
   - Check for smooth operation
   - Test bucket attachment/detachment

3. **Wheel motor test:**
   - Test each motor independently
   - Check for equal power
   - Test steering (differential speed)

4. **Integrated test:**
   - Test all systems together
   - Verify control response
   - Check for interference between systems

### Step 8.4: Load Testing

**Start with light loads and gradually increase**

1. **Lift capacity test:**
   - Begin with 100kg load
   - Increase gradually to rated capacity
   - Monitor cylinder pressure
   - Check for structural flex or failure

2. **Stability test:**
   - Test with loads at various heights
   - Verify machine doesn't tip
   - Test on level and sloped surfaces

3. **Operational test:**
   - Perform typical work tasks
   - Drive with loaded bucket
   - Test on various terrain

## Maintenance Schedule

### Daily (Before Operation)
- Visual inspection of frame and welds
- Check hydraulic fluid level
- Inspect hoses for leaks or damage
- Test all controls
- Check tire pressure and wear

### Weekly
- Lubricate pivot points
- Check bolt torque
- Inspect electrical connections
- Clean hydraulic filters

### Monthly
- Change hydraulic fluid (first change)
- Detailed inspection of welds
- Check for loose bolts
- Test emergency stop systems

### Annually
- Full hydraulic system service
- Repaint as needed
- Replace worn components
- Professional inspection recommended

## Troubleshooting

### Machine Won't Move
1. Check hydraulic fluid level
2. Verify pump is running
3. Check for hydraulic leaks
4. Test valve operation
5. Verify motor connections

### Loader Arms Move Slowly
1. Check hydraulic pressure
2. Inspect cylinder seals
3. Check for blocked hoses
4. Verify valve settings
5. Test cylinder for internal leaks

### Uneven Wheel Power
1. Check hydraulic flow to each motor
2. Inspect motor for damage
3. Verify hose sizes match
4. Check valve settings
5. Test motor displacement

### Control System Issues
Refer to `../../INSTALLATION_GUIDE.md` for electronics troubleshooting
