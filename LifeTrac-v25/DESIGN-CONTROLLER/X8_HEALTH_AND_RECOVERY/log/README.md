# Session log entries

One file per session per board. Naming convention:

```
YYYY-MM-DD_<board>_<routine_or_recovery>_vN.md
```

Example: `2026-05-13_board1_HC-01_v1.md`.

Keep entries short and structured (use the template below). Long-form
analysis still belongs in `LifeTrac-v25/AI NOTES/` — link to it from here.

## Template

```markdown
# YYYY-MM-DD <board> <routine_or_recovery>

- **Cycles / scope:** <e.g. 5 cycles>
- **Result:** PASS | FAIL | PARTIAL | UNREACHABLE
- **Evidence log:** [...](path)
- **Key observation:** one sentence
- **Action taken:** one sentence (or "none")
- **Next step:** one sentence
```
