# LifeTrac I

**Year:** 2008

## Description
Articulated steering, 4 wheel drive, wheel chains, dedicated 55 hp diesel engine. Limited roll-over protection.

## Features
- Articulated steering
- 4 wheel drive
- Wheel chains
- Dedicated 55 hp diesel engine
- Limited roll-over protection
